# 基本信息

* 型号: 3B+
* 操作系统: `UBUNTU 18.04.4 LTS (Bionic Beaver)`

# 网络配置（netplan）

* 配置文件路径:

  ​	    `/etc/netplan/50-cloud-init.yaml`

* 配置文件内容:

```yaml
# 配置静态IP
network:
    ethernets:
        enp0s3:
            dhcp4: false
            addresses: [192.168.0.201/24]
            gateway4: 192.168.0.1
            nameservers:
                addresses: [192.168.0.1, 8.8.8.8]
    version: 2
```

* 应用配置:

```bash
sudo netplan --debug apply
```
